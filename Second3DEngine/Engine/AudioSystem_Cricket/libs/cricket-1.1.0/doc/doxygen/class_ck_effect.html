float CkEffect::getWetDryRatio ( ) const

Gets the wet/dry ratio for this effect.

The ratio can range from 0 to 1. A value of 0 means the audio is left completely "dry", that is, with no effects applied; it is equivalent to bypassing the effect. A value of 1 means the audio is completely "wet", that is, the original "dry" signal is not mixed back into the processed audio.

bool CkEffect::isBypassed ( ) const

Gets whether this effect is bypassed.
