Enumerator:
kCkEffectParam_BitCrusher_BitResolution_Int 

Number of bits of resolution to keep in the input samples, in [1..24].

kCkEffectParam_BitCrusher_HoldMs_Float 

Milliseconds to hold each sample value.

Higher values result in more reduction in the effective sample rate. Values around 1 ms provide a reasonable effect.
